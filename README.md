# SonarJam · 54321

第一人称心理惊悚 · 黑暗探索 · 声呐显影。

> 你以为你在用工具看清世界。实际上是工具在用你确认它自己的正确性。

绝对黑暗中，声呐是唯一的感知手段——但这份黑暗不是地球的真相，是玩家携带的感知系统主动屏蔽光、把正常世界翻译成恐怖的结果。十章节 `54321 → 12345`，从依赖工具到识破牢笼。

**技术栈**：Rust + macroquad（Windows PC，48 小时 GameJam MVP）。

---

## 快速开始

```bash
cargo run
```

首次编译会拉取并构建 macroquad，约需 1–2 分钟。

### 控制

| 输入 | 动作 |
|------|------|
| `W` `A` `S` `D` | 移动 |
| 鼠标 | 视角 |
| 左键单击 | 发射声呐脉冲 |
| 左键长按 | 连续扫描 |
| `Esc` | 退出 |

发射消耗能量（左下能量条），默认不自动恢复，耗尽则无法发射。

## 目录结构

```text
GameJam/
├── README.md
├── Cargo.toml / Cargo.lock
├── docs/        # 设计与治理文档（见下方文档地图）
├── assets/      # ui/{gun,hud}、fonts —— 美术资产
└── src/
    ├── main.rs
    ├── app/     # 入口、配置、主循环编排
    ├── player/  # 第一人称相机、输入、移动碰撞
    ├── world/   # 真实关卡几何、碰撞、raycast —— 真实世界层（不可篡改）
    ├── sonar/   # 发射、采样、能量、持久点云 —— 感知结果层
    ├── render/  # 黑暗基底、点云、枪口细线
    └── ui/      # 锚点布局 + 可插拔元素的可扩展 UI 框架
```

**架构铁律**：`world`（真实几何）与 `sonar`/`render`（显影结果）严格分层。"工具有罪"靠感知层注入偏差实现，绝不篡改真实世界几何。
