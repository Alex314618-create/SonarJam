## 1. 目标

跑出核心声呐情绪原子，验证 macroquad 3D 可行性与手感：

> 绝对黑暗 → WASD + 鼠标第一人称移动 → 长按左键发射、枪口细线相连、命中点云持久显影、世界在深度遮挡下成形 → 能量条倒数 → 耗尽变黑。

手感权威参照：C 原型 `C:\Users\ROG\Desktop\sonar`（见 memory `sonar-feel-recipe`）。本切片用**代码定义几何**，不碰 GLB（RTB-005）。

## 2. 技术栈

- Rust + macroquad（RTB 已批准）
- 仅引入 macroquad 一个运行时依赖。引入任何其他 crate 前必须先问 PA（技术宪法）。

## 3. 模块与目录

```text
src/
  main.rs        # 入口，主循环，状态装配
  app/           # 窗口/全局生命周期/配置
  player/        # 第一人称相机、输入、移动、碰撞
  world/         # 真实几何（代码定义）、碰撞查询、raycast —— 真实世界层
  sonar/         # 发射规则、采样、能量、显影点生成 —— 感知结果层
  render/        # 黑暗基底、点云渲染、枪口细线
  ui/            # 可扩展 UI 元素系统
```

**架构铁律（主题命门，见 memory `guilty-tool-principle`）**：`world` 持有真实几何与碰撞，永远干净、不被篡改；`sonar`/`render` 产出显影结果。二者分层。本切片暂不实现偏差，但分层边界现在就要守住，后续"工具撒谎"是叠一层、不是返工。

## 4. 世界层（world）

- 代码定义一个走廊/房间几何：地面、天花板、若干墙体，构成一个有纵深、有拐角的可探索空间（足够让点云显影出"空间感"）。
- 提供 `raycast(origin, dir, max_range) -> Option<Hit{ pos, normal, surface_kind }>`，对世界三角形求交。
- 提供玩家移动用的碰撞查询（简单 AABB 或胶囊 vs 墙即可）。
- `surface_kind`：至少区分 Wall / Floor / Ceiling，供显影分色。

## 5. 感知层（sonar）

复刻 C 原型配方（数值可作为起点，后续可调）：

- **脉冲发射（左键单击）**：斐波那契螺旋锥，宽模式 ~120 射线 / 30°；每发消耗能量、有 cooldown。
- **连续发射（左键长按）**：随机角度锥，按 dt 累积射线速率持续消耗能量。
- 每条射线 `raycast` 命中后散射出墙点(+XZ 抖动) + 墙基地面点 + 天花板点，按 `surface_kind` 分色（墙青蓝 / 地面青 / 天花板浅蓝）。
- **持久点云**：环形缓冲（容量先取 65536），新点 push，缓冲满后覆盖最旧。点默认持久（不随时间消失）。
- 暴露"本帧新增点区间"（frame_start..write_head）供 render 画枪口细线。
- **能量**：上限固定、默认不自动恢复；耗尽则无法发射。

## 6. 渲染层（render）

- **黑暗基底**：清屏近黑 `(0.008, 0.008, 0.02)`；世界几何只写深度不写颜色（深度预通道），使世界不可见但正确遮挡点云。
- **点云**：加法混合，按距离调点大小（近 ~3px、远 ~1px），圆形点，满色。
- **枪口细线**：屏幕空间线段，从枪口锚点连到本帧每个新生点的屏幕投影；枪口端亮、命中端暗、加法混合。
- 第一人称相机投影矩阵供点云与细线共用。

## 7. UI 层（ui）—— PA 硬约束：不许嵌死（见 memory `ui-extensibility-mandate`）

必须做成可扩展元素系统，**不得把布局坐标/样式硬编码在渲染逻辑里**：

- **锚点布局**：每个元素挂在 `Anchor`（九宫格 TopLeft..BottomRight + Center）+ 像素偏移上；窗口尺寸变化时自动重排，不写绝对像素常量。
- **元素即插件**：定义统一 `UiElement` trait（如 `update(&mut self, ctx)` / `draw(&self, ctx)`）；`Ui` 持有 `Vec<Box<dyn UiElement>>`，遍历更新与绘制。新增 UI = 实现 trait + 注册一行，零侵入。
- **样式数据分离**：颜色/尺寸/字体集中在一处主题配置，不散落各元素。
- 本切片初始元素：能量条、枪部件（发射时高亮）、准星。为未来元素（叙事文本、系统告警、章节/死亡计数）预留：新增它们不应需要改 `Ui` 框架本身。

## 8. 输入与控制

- `WASD` 移动，鼠标看向（捕获鼠标），`Esc` 释放/退出。
- 左键：单击=脉冲，长按=连续发射。

## 9. 最小验证清单

1. `cargo run` 能编译并启动，出现一个近黑窗口。
2. 鼠标看向 + WASD 移动正常，撞墙被挡住。
3. 长按左键：枪口发出细线、命中处生成持久点云、世界轮廓在点云中浮现。
4. 单击左键：脉冲一次性显影。
5. 能量条随发射下降，耗尽后无法继续发射。
6. UI 元素按锚点定位；改变窗口大小时元素不错位、不写死。
7. 手感与 C 原型方向一致（黑暗中"打出去、世界短暂成形"的感觉成立）。

- 不做回溯/残留/EchoMode/叙事/音频
- 不做复杂物理、敌人 AI
